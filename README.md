# BeyondCTCI Submission Sync

This repository is managed by BeyondCTCI GitHub sync.

## What gets synced
- Only problem submissions (not code runs)
- Sync mode can be configured to:
  - Accepted only
  - All submissions

## File structure
- `DSA-problems/<problem-name>/<YYYY-MM-DD-HH-MM-SS-submission-id>.<ext>`

## Notes
- This is a one-way sync from BeyondCTCI to GitHub.
- Reconnecting GitHub creates a new repository target.
